PANTALLA PARA REALIZAR COMPRAS
==============================
- POR DEFECTO EL CAMPO DE CODIGO VA A ESTAR SELECCIONADO PARA CARGA RAPIDA CON LECTOR DE CODIGO DE BARRA
- '+' Y '-' PARA MODIFICAR LA CANTIDAD POR CADA RENGLON, POR DEFECTO MODIFICAR ULTIMO ITEM INGRESADO
    CUANDO ESTE SELECCIONADO EL CAMPO DE INGRESO DE CODIGO
- OPCION PARA QUE AUTOMATICAMENTE VAYA A LA PANTALLA DE AGREGAR ARTICULO PARA CODIGOS DE BARRA NO EXISTENTES.
    SI SE AGREGA UN ARTICULO DESDE COMPRA, PONER EL STOCK SIEMPRE EN CERO, NO DEJAR CAMBIARLO.
    EL STOCK SE VA A ACTUALIZAR EN BASE A LOS DATOS DE LA COMPRA, ASI QUE LOS ARTICULOS INGRESADOS DE ESTA MANERA
    TIENEN QUE ESTAR EN CERO PARA DEJAR BIEN REGISTRADA LA COMPRA Y TRANSACCION.
- SI EL CODIGO DE BARRA EXISTE, AGREGAR AUTOMATICAMENTE EL PRODUCTO.
- PERMITIR BUSCAR ARTICULOS, PANTALLA SIMILAR A CONSULTA DE ARTICULOS, PERO SOLO LECTURA.
- AÑADIR ITEMS SI SE INGRESA EL MISMO CODIGO VARIAS VECES
- ELIMINAR ITEMS CON CTRL+SUPR, PEDIR CONFIRMACION
- FINALIZAR COMPRA, PEDIR CONFIRMACION, PRESENTAR RESUMEN Y PEDIR SEGUNDA CONFIRMACION.

PANTALLA PARA REALIZAR SEÑAS
============================
- TABLA DE ARTICULOS IGUAL A LA DE VENTAS.
- EL MONTO TOTAL INGRESADO ES PARA TODA LA SEÑA, CON TODOS LOS ITEMS.
    ES DECIR, NO SE ASIGNAN MONTOS POR ITEM INDIVIDUALES.
- ES OBLIGATORIO PONER UN CLIENTE DISTINTO A 'CONSUMIDOR FINAL'.
- LA SEÑA AGREGA CREDITO AL CLIENTE. ESE CREDITO SE CANCELA CUANDO LA VENTA SEA PAGADA EN SU TOTALIDAD.

PANTALLA PARA REALIZAR RETIROS
==============================
- TABLA DE ARTICULOS IGUAL A LA DE VENTAS.
- SIN MONTOS, ESTO ES PARA RETIRO POR PARTE DEL DUEÑO
- TIENE UN ESTADO ASOCIADO, POR SI ESOS ITEMS SON RETIRADOS TEMPORAL O PERMANENTEMENTE.

SELECT ARTICULO.*, 'COMPRA', ITEM_COMPRA.CANTIDAD, COMPRA.FECHA_HORA
FROM ARTICULO
INNER JOIN ITEM_COMPRA
  ON ARTICULO.ARTICULO_ID = ITEM_COMPRA.ARTICULO_ID
INNER JOIN COMPRA
  ON ITEM_COMPRA.COMPRA_ID = COMPRA.COMPRA_ID
WHERE COMPRA.ANULADA = 0
    
SELECT ARTICULO.*, 'VENTA', ITEM_FACTURA.CANTIDAD, ITEM_FACTURA.PRECIO, FACTURA.FECHA_HORA
FROM ARTICULO
INNER JOIN ITEM_FACTURA
  ON ARTICULO.ARTICULO_ID = ITEM_FACTURA.ARTICULO_ID
INNER JOIN FACTURA
  ON ITEM_FACTURA.FACTURA_ID = FACTURA.FACTURA_ID
WHERE FACTURA.ANULADA = 0

SELECT ARTICULO.*, 'RETIRO', RETIRO_ITEM.CANTIDAD, RETIRO_ITEM.DETALLE, RETIRO_ITEM.FECHA_HORA, RETIRO_ITEM.ESTADO
FROM ARTICULO
INNER JOIN RETIRO_ITEM
  ON ARTICULO.ARTICULO_ID = RETIRO_ITEM.ARTICULO_ID

SELECT ARTICULO.*, 'SEÑA', ITEM_SENIA.CANTIDAD, SENIA.DETALLE, SENIA.FECHA_HORA, SENIA.ESTADO, CLIENTE.NOMBRE
FROM ARTICULO
INNER JOIN ITEM_SENIA
  ON ARTICULO.ARTICULO_ID = ITEM_SENIA.ARTICULO_ID
INNER JOIN SENIA
  ON ITEM_SENIA.SENIA_ID = SENIA.SENIA_ID
INNER JOIN CLIENTE
  ON SENIA.CLIENTE_ID = CLIENTE.CLIENTE_ID
